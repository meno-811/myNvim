个人手搓的Nvim配置项目，凝结了我深深的感情（bushi
目前只支持写go，python，lua

需要Nvim版本大于0.12
Nvim的下载方式需要自己检索

安装方式：

''' bash
git clone https://github.com/meno-811/myNvim.git ~/.config/nvim
'''
 
在nvim里通过:Mason下载go/python/lua的LSP服务


快捷键设置：
空格+e 打开文件树插件
空格+数字 打开文件树插件的子窗口
ctrl+h/j/k/l 移动到左/下/上/右窗口
ctrl+上下左右 调整窗口大小
ctrl+\ 打开终端
